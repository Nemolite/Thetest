

- Функция prepare_xml() - предназначена для передачи данных из формы на сервер используя AJAX (строка 85 script.js раскомментировать). Работоспособность была проверена на OpenServer. (обработчик controller.php)

(Выполнение кода можно посмотреть http://jstest.uxp.ru/)

- Функция prepare_websocket() - для передачи данных через веб-сокет (строка 85 script) . Для проверки работоспособности нужно запустить серевер командой node server.js. (Передварительно необходимо установить ws -> https://github.com/websockets/ws)

- Функция out_data() - для вывода данных полученных через AJAX

- Функции valid_input(), valid_age() - для валидации полей формы.

- Функция transport_from_server() - для обновления баланса через веб-сокет.

- Функция ajax_transport_from_server() - для обновления баланса через AJAX
